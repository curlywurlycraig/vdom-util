vdom-util

A demo of vdom-util. If React and jQuery had a child, this would be it.

This toolkit is really useful if you have some HTML that you're particularly fond of, and you just want to sprinkle in some functionality on top.

It's also great if you like vanilla JS but ocassionally want a few helpers.

JS is required for this page to work correctly.
